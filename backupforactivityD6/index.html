I think so too. In fact, here is an unordered list of reasons why I think HTML is awesome:

  • HTML can be edited in real time. It doesn't have to be recompiled into an executable, like C++ or some other languages.
  • HTML doesn't freak out and entirely break when you make a small mistake. It just ignores that you were stupid and displays all of the parts of your code that actually do work.
  • HTML can be edited with CSS and Javascript, to the point where there are no limits to what you can or cannot do!
